Introduction to Mohaaseb

Mohaaseb is a comprehensive Cloud ERP built to simplify business management. It unifies accounting, HR, and inventory into a single, ZATCA-compliant platform.

Note regarding Phase 2 Compliance

Mohaaseb automatically handles the cryptographic stamping required for ZATCA Phase 2. No manual configuration is needed.

System Requirements

Since Mohaaseb is cloud-based, there are no hardware installations required. However, for the best experience:

  • Modern Web Browser (Chrome 90+, Edge, Firefox, Safari).
  • Stable Internet Connection (10 Mbps recommended).
  • For POS: A thermal printer supporting QR codes.

API Quickstart

Developers can integrate with Mohaaseb using our REST API. Here is a simple example to fetch your profile:

Terminal 
curl -X GET https://api.mohaaseb.com/v1/profile \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
